# Day 45 - Advanced Bash Scripting (Automation Foundations)

## Objective
To enhance Bash scripting skills by introducing command-line arguments and system environment variables for building reusable and system-aware automation scripts.

---

## What I Implemented

### 1. Command-Line Arguments Script
- Captured dynamic inputs using $1, $2
- Handled multiple arguments using $# and $@
- Built reusable script structure for automation tasks

### 2. System Variables Script
- Extracted system information such as:
  - Current user
  - Home directory
  - Working directory
  - Shell environment
  - Hostname
  - Date and time

---

## Key DevOps Concepts

- Script parameterization
- Environment awareness
- Automation readiness
- System context handling
- Debugging environment variables

---

## Scripts Created

- args.sh → Handles command-line arguments
- system_vars.sh → Displays system environment details
