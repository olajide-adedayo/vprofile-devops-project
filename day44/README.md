# Day 44 - Bash Scripting Fundamentals (Automation Basics)

## Objective
To learn and practice Bash scripting fundamentals including variables, user input, and loops for basic automation tasks in DevOps.

---

## What I Learned

- How to use variables in Bash (user-defined and system variables)
- How to accept user input using read
- How to implement conditional logic (if/else)
- How to use loops (for and while) for automation

---

## Scripts Developed

- variables.sh → Demonstrates variables and system environment
- user_input.sh → Interactive script using user input
- loops.sh → Demonstrates for and while loops

---

## Key DevOps Concepts

- Automation using scripting
- Dynamic script execution
- Basic control flow in Bash
- Reusable script design
